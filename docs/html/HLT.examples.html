HLT.examples.main module

HLT.examples.main.analyze_circ(sys_size, circ, qubits_to_reconstruct, backend, range_constraints, range_hamiltonians, num_parameters, total_number_of_shots, optimization_level=3, initial_layout=None, is_torch=False)
Parameters

  • sys_size (int) – number of qubits in the circuits

  • circ (qiskit.circuit.quantumcircuit.QuantumCircuit) – circuits which produces the wanted density matrix

  • qubits_to_reconstruct (List[int]) – which qubits to reconstruct the density matrix for with HLT

  • backend (Union[qiskit.providers.aer.backends.aer_simulator.AerSimulator, qiskit.providers.backend.Backend]) – which backend to measure expectation values with.

  • range_constraints (int) – the maximum locality of the constraints, which are taken to be Pauli operators, in the constraint matrix (should be bigger than range_hamil_terms, usually taken as range_constraints=range_hamil_terms+1).

  • range_hamiltonians (int) – the maximum locality of the Hamiltonians which assumed to be in the GH. These Hamiltonian will define the constraint matrix.

  • num_parameters (int) – number of parameters to used in HLT ansatz (denoted as l in the main text graphs).

  • total_number_of_shots (int) – total number of shots to use to measure the expectation values with.

  • optimization_level (int) – How much optimization to perform on the circuits, as described in qiskit.compiler.transpile

  • initial_layout (Optional[List[int]]) – Initial position of virtual qubits on physical qubits, as described in qiskit.compiler.transpile

  • is_torch (bool) – whether to use PyTorch optimization or SciPy optimization, see main text for more details

Returns

an AnsatzResults with the following data: the cyclic measurer used for HLT, the ansatz produced by HLT, the final loss of HLT optimization, the HLT reconstructed density matrix, the ground truth density matrix calculated with Qiskit simulation and the fidelity between the reconstructed density matrix and the ground truth density matrix (can also be saved in Pickle file).

Return type

tomography.ansatz.AnsatzResults
